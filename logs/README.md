# Performance Logs

This directory contains performance metrics generated by `src/lib/perf.ts`.

## Files

- `perf.ndjson` - Newline-delimited JSON logs of all API requests

## Log Format

Each line is a JSON object with:
```json
{
  "route": "admin/messages",
  "requestId": "req_1234567890_abc123",
  "msTotal": 150,
  "status": 200,
  "bytesOut": 25600,
  "dbCalls": 3,
  "dbMsTotal": 85,
  "timestamp": "2026-01-19T12:00:00.000Z"
}
```

## Analysis

To analyze logs, use:
```typescript
import { analyzePerfLogs } from '@/lib/perf'
console.log(analyzePerfLogs())
```

Or use scripts:
```bash
./scripts/perf/analyze.sh
```
